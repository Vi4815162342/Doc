Il faut maintenant extraire le fichier

tar xzfv wordpress-4.7.3-fr_FR.tar.gz

Ensuite il faut déplacer le dossier dans le dossier /var/www/html/.. pour Debian Linux

cp -r ~/tmp/wordpress /var/www/html/wordpress

Maintenant on va s'attribuer les droits

sudo chmod 2775 -R /var/www/html/wordpress/wp-content/

On remplit nos données personelles

La taxonomie c'est quoi ?

Fondamentalement, une taxonomie est un moyen de regrouper les choses.

Par exemple, je pourrais avoir un tas de différents types d’animaux. Je peux les regrouper selon diverses caractéristiques, puis attribuer des noms de groupes.

C’est quelque chose que la plupart des gens rencontrent dans les cours de biologie, connu sous le nom de taxonomie linnéenne ou plus simplement classification classique/traditionnelle.

Donc ici ce sera la classification de plusieurs contenus !

Exemple : Destination, type de séjour...

Les termes.

Ce sont des éléments de classification

Exemple : Lyon, Paris = Taxonomie destination / voyage, croisiere = Taxonomie type de séjour

Les champs.

ce sont des éléments particulié pour des contenus ou terme avec plusieurs types de champ
